## Tech stack

Expo SDK 54 • React Native 0.81 • React Native Paper • TypeScript • Expo Router • Epilogue font

## Setup

```bash
bun install
bun run start
```

Also works with `npm install` / `npm run start`.

For Expo Go on a physical device, use `--tunnel` if not on the same network.
